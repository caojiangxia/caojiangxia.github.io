  • wq 表示保存并退出
  • q! 强制退出并忽略所有更改
  • e! 放弃所有修改，并打开原来的文件

文件的复制

复制粘贴文件： cp [参数] 源文件或者文件夹 目标文件或者文件夹

简单来说：cp [option] source1 source2 source3 … directory

其中option有很多种：

  • a:是指archive的意思，也说是指复制所有的目录
  • d:若源文件为连接文件(link file)，则复制连接文件属性而非文件本身
  • f:强制(force)，若有重复或其它疑问时，不会询问用户，而强制复制
  • i:若目标文件(destination)已存在，在覆盖时会先询问是否真的操作
  • l:建立硬连接(hard link)的连接文件，而非复制文件本身
  • p:与文件的属性一起复制，而非使用默认属性
  • r:递归复制，用于目录的复制操作
  • s:复制成符号连接文件(symbolic link)，即“快捷方式”文件
  • u:若目标文件比源文件旧，更新目标文件

例如将A目录下的a文件复制到B目录，并将其改名为b，我们可以这样做

文件的剪切

剪切粘贴文件： mv [参数] 源文件或者目录 目标文件或者文件夹

简单来说：mv [option] source destination

其中option也比较简单，只有三种

  • f:force，强制直接移动而不询问
  • i:若目标文件(destination)已经存在，就会询问是否覆盖
  • u:若目标文件已经存在，且源文件比较新，才会更新

例如将A目录下的a文件剪切到B目录，并将其改名为b，我们可以这样做

修改文件的访问权限

chmod表示的是给文件分配权限。777表示的是三组参数，下面分别给出三个数字的含义

  1. 文件所有者的权限
  2. 文件所有者同一个用户组的用户权限
  3. 其他用户组及其组员的权限

与此同时，每个数字表示不同的含义，因为我们的操作总共有3种，分别为可读r，可写w，可执行x。三位表示的是二进制的三位1,2,4，相加的话就是7。换句话说，从0-7分别表示所拥有的不同权限。

举个例子来说

  • r-x表示的是4+1，二进制就是101，值就是5

    所以chmod 751 my file，这个语句对应的就是所有者可读可写可执行，同用户组的用户可以读可执行，其他用户可执行。
